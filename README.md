# Pantum BP5100DW Advanced Test Stand

Профессиональный инженерный стенд для тестирования и оптимизации высокопроизводительной печати на принтерах **Pantum BP5100DW** (и аналогичных серий BM/BP). 

Спроектирован для разработчиков киосков самообслуживания, систем автоматизации и высоконагруженных систем печати.

## 🚀 Ключевые возможности

### 1. Методы сверхбыстрой печати
*   **Native PDF (TCP 9100)**: Прямая отправка PDF-данных в сокет принтера с PJL-заголовками. Самый быстрый способ, полностью обходящий драйверы Windows.
*   **Win32 RAW Spooling**: Отправка данных напрямую в очередь Windows Spooler в формате RAW.
*   **PJL Job Handling**: Именование заданий, управление копиями, разрешением и режимами плотности.

### 2. Оптимизация производительности (Speed Hacks)
*   **PDF Flattening (PyMuPDF)**: «Сплющивание» сложного PDF в растр на стороне ПК с настраиваемым DPI (300/600/1200).
*   **Smart Fuser Pre-heat**: Прогрев печки без печати листа через PJL-команду `@PJL INFO ID`.
*   **PDF Auto-Repair**: Автоматическая очистка и разблокировка PDF перед отправкой (`clean=True, deflate=True`).
*   **Smart Method Selection**: ИИ-рекомендация оптимального метода печати на основе размера файла.
*   **DPI Control**: Переключатель 300/600/1200 DPI для баланса скорость/качество.

### 3. Диагностика и управление
*   **SNMP Monitoring**: Тонер, ресурс барабана, счетчик страниц, ошибки — в реальном времени.
*   **Ping Tracker**: Индикатор задержки сети до принтера (зелёный/красный).
*   **Deep Health Check**: Проверка всех портов (80, 9100, 161, 515, 631) за один клик.
*   **Auto Discovery + Bonjour/mDNS**: Поиск принтеров по TCP и протоколу Apple Bonjour.
*   **Benchmarking**: Автоматическое логгирование в `print_stats.csv`.

### 4. Удалённое управление (PJL Remote Control)
*   **📟 LCD Message**: Кастомный текст на дисплее принтера (до 16 символов).
*   **🛑 Cancel Job**: Экстренная отмена текущего задания печати.
*   **🔄 Remote Reboot**: Удалённая перезагрузка принтера через PJL INITIALIZE.

## 📦 Установка

1. Убедитесь, что у вас установлен Python 3.10+.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## 🛠 Использование

```bash
python gui_tester.py
```

### Основные шаги:
1. Выберите файл (PDF, DOCX, TXT или изображение).
2. Нажмите **🔍 Найти принтеры** или введите IP вручную.
3. (Опционально) Нажмите **🔥 Прогрев печки** для подготовки фьюзера.
4. Выберите **DPI** и опцию **Сплющивать PDF** при необходимости.
5. Нажмите **🚀 ЗАПУСТИТЬ ТЕСТИРОВАНИЕ**.

## ⚙️ Гайд по настройке (Speed Tuning)

Для получения времени выхода первого листа **менее 3 секунд**:

1. **В Windows**:
   - Свойства принтера -> Дополнительно -> **«Печатать прямо на принтер»**.
2. **В веб-интерфейсе принтера (Pantum Web UI)**:
   - Отключите **Тихий режим (Quiet Mode)**.
   - Отключите **Эко-режим (Eco Mode)**.
   - Установите таймер сна на максимум (120 минут).
3. **В приложении**:
   - Используйте **300 DPI** для документов — ускорение передачи в 4x.
   - Включите **Flattening** для сложных PDF с графикой.

## 📁 Структура проекта
*   `gui_tester.py` — основной файл приложения (GUI).
*   `core/printer.py` — ядро: FastPrinter, PJL, flattening, wake-up.
*   `core/network.py` — TCP/SNMP/Health Check/Discovery.
*   `core/converters.py` — конвертеры форматов (PDF, DOCX, IMG, TXT).
*   `core/fallback.py` — цепочка fallback-методов (SumatraPDF, Win32, Adobe).
*   `requirements.txt` — зависимости.
